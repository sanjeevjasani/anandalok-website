# Anandalok Website — Copy Changes Log

**Reason for changes:** Client feedback — Anandalok serves individuals with autism, Down syndrome, ADHD, epilepsy, and other intellectual disabilities. The website was previously too narrowly focused on autism alone. All copy has been updated to reflect the full breadth of conditions the organisation supports.
